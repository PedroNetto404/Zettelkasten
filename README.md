# Zettelkasten - Caixa de anotações 
## O que você encontra aqui? 
Aqui você encontra minhas anotações de aulas, livros, podcasts e etc. Todo material que consumo tento extrarir os tópicos mais importantes e amadurece-los atravès de um workflow composto de 3 etapas:
1. Fleeting Notes: notas em que capturo ideias, citações e etc; 
2. Literature Notes: Elabora um breve resumo proveniente das notas fugazes tentanto ao máximo escrever o meu entendimento sobre os objetos de estudo com as minhas próprias palavras; 
3. Permanent Notes: São notas permanentes que contém em si uma úncia ideia de maneira atômica. Essas notas expressão meu entendimento a respeito das ideias estudadas. Com o tempo, vou ligadando ideias de maneira bidirecional. São links naturais e organicos. Essas notas servem de material para elaboração de roteiros para o meu canal do youtube e de material para revisões periódicas em sistemas de repetição espaçada, como o Anki.

## Organização física: 
 O método por si só não impõe nenhum requisito para a forma como armazenamos nossas notas. Prefiro separa-las em pastas com seu tipo correspondente.
