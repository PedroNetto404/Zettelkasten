# Workflow
## Fleeting Notes (Notas fugazes)
![[📂 Files/🖼️ Images/Pasted image 20221007151700.png]]
+ São notas que não ficam muito tempo paradas, pois nelas queremos apenas capturar as ideias para que não se percam. Posterirormente vamos destrinçar isso melhor; 
+ Ideias; 
+ Citações; 
+ Destaques de livros; 
+ Captura inicial de pensamentos; 
	+ Tirar para fora da cabeça um pensamento que pode se perder; 
+  São fontes para as literature notes
## Literature Notes (Notas literárias)
![[📂 Files/🖼️ Images/Pasted image 20221007151642.png]]
+ Representa a minha compreensão sobre um determinado assunto; 
+ Aqui escrevo o que essas ideias significam para mim, com as minhas palavras. 
+ É um resumo de meu entendimento sobre um assunto
## Permanent Notes
![[📂 Files/🖼️ Images/Pasted image 20221007151846.png]]
+ Ideia única em uma nota
+ Deve ser fácil de entender isoladamente
	+ Posso pegar qualquer nota e compreender o que ela significa (mesmo que eu leia essa nota daqui anos)
	+ Não há parafaseamento ou cópia literal de algo
		+ Cada nota isolada de uma ideia representa meu entendimento sobre a mesma ideia
## Insights
+ Podemos ligar bidirecionalmente para criar um entendimento
+ ![[📂 Files/🖼️ Images/Pasted image 20221007152541.png]]
+ No exemplo temos duas ideia ligadas para criar uma nova percepção sobre o cenário de estudo; 
+ Por outro lado, podemos elaborar questionamento e analisa-los a partir das ideias atômicas armanezadas nas notas

## Conclusão
1. Fleeting Notes $\rightarrow$ Capturar as ideias; 
2. Literature Notes $\rightarrow$ Criar meu próprio entendimento para resumir isso em literature notes; 
3. Permanet Notes $\rightarrow$ Organizar as ideias em notas permanentes
4. Vircular permanente notes para criar insights
