# Os tipos de uma associação - agregação, composição e dependência
+ Duas formas
	+ Estrutural
		+ Ocorre na estrutura de dados da classe.
		+ Um dos atributos de uma classe é do tipo de outra classse.
		+ Agregação 
			+ Uma parte do objeto em uma associação estrutural que pode estar sendo compartilhada por vários objetos
		+ Composição
			+ Uma classe só existe quando composta a outra.
		+ Os dois tipos podem ter uma implementação igual, mas conceitualmente são diferentes
	+ Comportamental
		+ Dependência
