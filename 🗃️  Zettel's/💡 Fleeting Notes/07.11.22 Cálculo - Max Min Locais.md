# 07.11.22 Cálculo - Max Min Locais

Seja f uma função e p um ponto de seu domínio. O ponto p é dito ponto máximo local de f se f(x) <= f(p) para todo x em uma vizinhaça de p. Analogamente, p é chamdo ponto de mínimo local de f se f(x) >= f(p) para todo x em uma vizinhança de p. Analogamente, p é chamado ponto mínimo local de f se f(x) >= f(p) para todo x em uma vizinhança de p. p é chamado de ponto crítico de f se f'(p) = 0 ou f'(p) não existe. 

**Teorema:**
Seja f uma função e p um ponto de seu domínio. Se p é ponto de máximo ou mínimo local de f, então p é ponto crítico de f. 

**Teorema:**
Seja f uma função que possui derivadas contínuas ao menos até a 2ª ordem em uma vizinhaça de p, com p pertencendo ao domínio de f. 

i) f'(p) = 0 e f''(p) > 0 => p é ponto de mínimo local de f;

ii) f'(p) = 0 e f''(p) < 0 => p é ponto de máximo local de f.
