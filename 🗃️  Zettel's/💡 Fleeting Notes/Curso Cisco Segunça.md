# Curso Cisco Segunça

## O que é a segurança cibernética
+ Contexto de redes conectadas de transporte de informações digitais
	+ Como proteger esse sistema de rede de maneira eficiente? 
	+ Proteger 
		+ identidade
		+ Dados 
		+ Dispositivos
		+ Dados de clientes

## Identidade on-line e off-line
+ Nossa identidade off-line é como somos como pessoa que interage diariamente com amigos e familiares, em casa, na escola ou trabalho.
	+ Esses pessoas conhecem nossas informações pessoais
		+ Nome
		+ Idade
		+ Onde moramos
+ Nossa identidade on-line é sobre como somos no espaço cibernético.
	+ Como nos apresentamos aos outros on-line
	+ Deve revelar uma quantidade limitada de informações sobre nós
	+ Nome de usuário
		+ alias para identidades on-line
		+ Não deve incluir informações pessoais
		+ Algo apropriado e respeitoso
		+ Não deve transparecer que somos alvos fáceis para crimes cibernéticos ou atenção indesejada.
## Nossos dados

		+ médicas

		+ Servidores localizados em partes diferentes do mundo
