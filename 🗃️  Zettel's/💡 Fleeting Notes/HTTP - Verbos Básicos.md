#  HTTP - Verbos Básicos
Protocolo HTTP está presente em todo o ciclo de vida de uma aplicação web; 
Tudo é baseado no que o protocolo HTTP permite;
## Verbos HTTP
### Get
+ Semelhente ao select do CRUD
+ Pede (request) uma informação ao server; 
+ É feito através da URL;
### Post
+ Semelhante ao insert do CRUD
+ Representa uma intenção
+ Enviando informações ao servidor (formulários)
### Put
+ Semelhante ao update do CRUD
Similar ao Post, envia informações ao servidor; 
É utilizado para atualizar informações existentes
### Delete
+ Solicita a exclusão de uma informação no servidor através da URL indicada
