# Contas contábies e plano de contas
+ A contabilidade para registrar os fatos contábeis se utiliza de contas 
+ Contas são termos técnicos para representar componentes patrimoniais e de resultado
## Patrimoniais
+ Contas registradas no balanço patrimonial
### Contas de ativo
+ Naturezas devedoras
### Contas de passivo
+ Naturezas credoras
## Resultado
### Contas de receita
+ Naturezas credoras
### Contas de despesas
+ Naturezas devedoras

+ Cada lançamento é registrado em um plano de contas
## Plano de contas
+ É onde encontramos as contas contábeis para fazer os débitos e créditos
### Contas sintéticas
+ Não recebem lançamento
+ Recebem o saldo das contas analíticas que fazem parte dela
### Contas analíticas
+ São contas mais abaixo na categorização das contas sintéticas

#### Quando formos registrarmos um conta
+ Utilizamos as contas analíticas
+ Contas sintéticas recebem o saldos das contas analíticas
+ Para construir o plano de contas
	+ Código
	+ Nome de cada conta
	+ Natureza (devedora ou credora)
	+ Informação sintética ou analítica
