# ASP.NET Core Enterprise Application With Eduardo Pires

## Objetivo
+ Desenvolver uma aplicação estilo e-commerce utilizando conceitos avançados do ASP.NET Core e de modelagem. 
+ Abordar diversos cenário de aplicações corporativas onde o conhecimento e planejamento são fundamentais para implementação de uma solução sólida e bem estruturada. 
## Enterprise application
+ Aplicação com propósito comercial
	+ Oferecer uma solução
	+ Gerenciar uma operação
	+ Automatizar um processo
	+ Fornecer um produto / Serviço digital
	+ Realizar integrações
+ Implica desenvolver aplicações
	+ Atendem um setor comercial
	+ Atendem uma regra de negócio
	+ Implementar características do mundo real
+ O que define a arquitetura da aplicação 
	+ Desafios
	+ Complexidade que deverá ser aplicada para entregar uma aplicação que atenda todas as necessidades do negócio em todos os cenários possíveis
+ Requer uma série de preocupações
	+ Facilidade de mudanças  (modelagem)
	+ Estabilidade
	+ Escalabilidade
	+ Performance
		+ Deve ser feita do jeito certo
			+ Comunicação com o banco
			+ Correta implementação para uso correto dos recursos
	+ Segurança
		+ Dados sensíveis
		+ App não deve ser burlável
	+ Baixa complexidade
		+ A menor possível
