# OOP Eduardo Pires

+ Em POO devemos sempre buscar o baixo acoplamento e a alta coesão entre objetos
+ **Acoplamento**
	+ Dependência direta entre um objeto e outro.
	+ Quando um objeto muda o outro muda por consequência
+ **Coesão**
	+ Objetos executando uma única responsabilidade livres de uma depêndencia direta com um outro objeto
+ **Classe**
	+  Estrutura que encapsula uma estrutura de dados e comportamentos que agem em cima desses.
	+ O intuito de uma classe é representar um conceito do mundo real
+ **Objeto**
