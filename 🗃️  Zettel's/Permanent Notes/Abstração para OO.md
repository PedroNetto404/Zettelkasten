# # Conceito: Abstração para OO

Abstração é o ato de olhar para o mundo real e abstrair seus objetos em representações que carregam em sí características relevantes para o problema analisado, deixando de lado informações irrelevantes. 
Uma abstração serve de base para a definição de classes em OO. Uma boa abstração nos leva a realizar um bom modelo de nossas entidades de negócio e ter um sistema mais flexível e fácil de manter.
