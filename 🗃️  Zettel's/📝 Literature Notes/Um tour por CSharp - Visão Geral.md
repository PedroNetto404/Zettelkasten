# Um tour por CSharp - Visão Geral

CSharp é uma linguagem fortemente tipada, bem atual e orientada a objetos.

Os principais recursos do charp são: 
Garbage Collector: 
+ É uma espécie de entidade que verificar quais objetos alocados em memória não estão sendo utilizados e quais precisam ser descartados para que a memória seja liberada. 
Tipos Anuláveis T: 
+ Representa todos os valores possíveis para um tipo T seguido de um valor adicional null; 
Tratamento de Exceções
+ Com CSharp temos ferramentas para tratar exceções que podem ocorrer em tempo de execução. 
Expressões lambdas
+ Expressõs lambda são sintaxes para a definição de funções locais podendo ser funções com retorno simples ou funções de bloco. 
Consulta integrada a linguagem (LINQ)
+ Conjunto de tecnologias
+ Temos uma sintaxe para operar sobre fonte de dados
Operações assíncronas
Sistema de tipos unificado
+ Em Csharp todo tipo seja os primitivos ou os definidos pelo usuário derivam de object, logo todos esses tipos compartilham um conjunto de operações comuns. 
Temos tipos de valor e tipos de referência que podem ser definidos pelo usuario
Suporte a entidades genéricas, seja ela uma classe, interface ou método.
Iterators
+ Implementadores de classes de coleção podem oferecer comportamentos personalidados para as classes clientes.
